== Description ==

This Google Buzz plugin allows you to display your latest Google Buzz updates as a widget on your blog!

Simply enable the plugin through your admin panel, edit the JR GoogleBuzz settings in the Settings --> JR GoogleBuzz menu and enable the widget through the Appearance --> Widgets menu! They will now be displayed on your blog!

== Installation ==

1. Upload `jr-googlebuzz.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR GoogleBuzz Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets menu.
